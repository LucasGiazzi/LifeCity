## Como usar no Overleaf

1. Faça upload de todos os arquivos para um projeto no Overleaf
2. Defina `main.tex` como documento principal
3. Adicione as imagens na pasta `images/`
4. Compile com pdflatex (2x para o sumário ficar correto)

## Integração com GitHub (opcional)
