## Sobre o projeto

O **LifeCity** é um aplicativo mobile que conecta moradores e poder público, permitindo que cidadãos registrem problemas urbanos e eventos da cidade diretamente em um mapa interativo.

- Reporte buracos, falta de iluminação, problemas de segurança e mais
- Descubra eventos locais: festas, esportes, educação, cultura
- Tudo geolocalizado em um mapa colaborativo da cidade

## Stack

| Camada | Tecnologia |
|---|---|
| Mobile | Flutter / Dart |
| Backend | Node.js + Express |
| Banco de dados | PostgreSQL (Supabase) |
| Storage | Supabase Storage |
| Autenticação | JWT (access token + refresh token) |
| Mapa | OpenStreetMap via flutter_map |

## Funcionalidades

- Mapa interativo com pins de eventos e reclamações
- Filtros por categoria (Infraestrutura, Segurança, Limpeza, Trânsito, Festas, Esportes, Educação)
- Criação de reclamações com foto, categoria e geolocalização
- Criação de eventos com data, hora e localização
- Autenticação JWT com renovação automática de sessão
- Modo claro e modo escuro
- Perfil com foto, edição de dados, histórico de publicações
